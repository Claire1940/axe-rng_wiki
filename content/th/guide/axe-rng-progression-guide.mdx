- สาธิตกลไกการตัดต้นไม้แบบไม่จำกัดในโซนเริ่มต้น
- แสดงความแตกต่างทางสายตาระหว่างการสุ่มขวานทั่วไปและหายาก
- อธิบายเค้าโครง UI พื้นฐานสำหรับผู้เล่นใหม่ที่เข้าสู่ป่า
- เน้นความเร็วของความก้าวหน้าเมื่อใช้โค้ดที่ใช้งานอยู่

| ชื่อการอัปเกรด | ลำดับความสำคัญ | ผลกระทบต่อการเล่นเกม |

| **ประสิทธิภาพการตัด** | **สูงสุด** | ลดจำนวนการโจมตีที่จำเป็นต่อต้นไม้ |
| **ผู้ค้นหาคริสตัล** | **สูง** | เพิ่มอัตราสกุลเงินการอัปเกรด |
| **ความจุท่อนไม้** | **ปานกลาง** | ช่วยให้ฟาร์มแบบ AFK ได้นานขึ้น |
| **โชคในการสุ่ม** | **สูง** | ปรับปรุงโอกาสความหายากของขวานโดยตรง |
| **ความเร็วผึ้ง** | **ต่ำ** | มีประโยชน์เฉพาะเมื่อคุณมีผึ้ง 10+ ตัว |

ในขณะที่คุณกำลังยุ่งกับการเหวี่ยงขวาน ผึ้งของคุณคือฮีโร่ที่ไม่ได้รับการยกย่องในบัญชีของคุณ ระบบผึ้งทำหน้าที่เป็นเส้นทางความก้าวหน้าคู่ขนาน น้ำผึ้ง ซึ่งเป็นทรัพยากรที่ผลิตโดยผึ้ง ใช้สำหรับการอัปเกรดพิเศษที่ท่อนไม้และคริสตัลไม่สามารถซื้อได้ **คู่มือความก้าวหน้า Axe RNG** ของเราแนะนำให้ปฏิบัติต่อผึ้งเสมือน "กองทุนเกษียณ" ของคุณ — ลงทุนในพวกมันตั้งแต่เนิ่นๆ และพวกมันจะช่วยคุณผ่านการบดขยี้ช่วงท้ายเกม

**วงจรความก้าวหน้าของผึ้ง:**
1. **ฟักไข่**: ใช้น้ำผึ้งหรือโทเค็นพิเศษเพื่อรับผึ้งใหม่
2. **เก็บเกี่ยว**: ผึ้งจะผลิตน้ำผึ้งโดยอัตโนมัติตลอดเวลา
3. **อัปเกรด**: ใช้น้ำผึ้งเพื่อเพิ่มความเร็วในการบินของผึ้งและปริมาณการเก็บเกี่ยว
4. **ขยาย**: ปลดล็อกช่องรังผึ้งเพิ่มเติมเพื่อเพิ่มรายได้พาสซีฟรวมของคุณ

| ความหายากของผึ้ง | อัตราน้ำผึ้ง | ความสามารถพิเศษ |
| :--- | :--- | :--- |
| **ผึ้งทั่วไป** | 10/นาที | ไม่มี |
| **ผึ้งงาน** | 25/นาที | เก็บเกี่ยวท่อนไม้เร็วขึ้น 5% |
| **ผึ้งนางพญา** | 100/นาที | เพิ่มพลังให้ผึ้งใกล้เคียง 10% |
| **ผึ้งนำโชค** | 50/นาที | เพิ่มโชคแบบคงที่ +0.5 ให้กับการสุ่ม |

<Callout type="tip" title="เคล็ดลับรายได้พาสซีฟ">
หากคุณวางแผนที่จะปล่อยเกมทิ้งไว้ข้ามคืน (AFK) ตรวจสอบให้แน่ใจว่า "ความจุท่อนไม้" ของคุณสูงพอที่จะรองรับการรับเข้า แต่ที่สำคัญกว่านั้น ตรวจสอบให้แน่ใจว่าผึ้งของคุณได้รับการอัปเกรด การเก็บน้ำผึ้งไม่ถูกจำกัดด้วยพื้นที่คลัง ทำให้เป็นทรัพยากร AFK ที่สมบูรณ์แบบ
</Callout>

<Section icon={Shield} title="กลไกการเกิดใหม่และเส้นทางโซน">

ในที่สุด คุณจะมาถึงจุดที่ต้นไม้ใช้เวลานานเกินไปในการตัดและการอัปเกรดมีราคาแพงเกินไป นี่คือสัญญาณให้เกิดใหม่ การเกิดใหม่จะรีเซ็ตท่อนไม้ ทักษะ และความคืบหน้าในโซนของคุณ แต่จะให้ตัวคูณถาวรแก่ **โชคพื้นฐาน** ของคุณ นี่เป็นวิธีเดียวที่จะสุ่มขวานระดับ S-Tier และ God-Tier ได้อย่างสม่ำเสมอในช่วงท้ายเกม

**เส้นทางความก้าวหน้าของโซน:**
- **โซน 1 (ป่าเริ่มต้น)**: มุ่งเน้นไปที่ขวานโค้ด THANKYOU และทักษะพื้นฐาน
- **โซน 2 (Autumn Grove)**: อุปสรรคสำคัญแรก ต้องใช้พลังงานอย่างน้อย 500
- **โซน 3 (Crystal Caves)**: ให้ผลผลิตคริสตัลสูง พักที่นี่เพื่อเพิ่มทักษะเริ่มต้นให้สูงสุด
- **โซน 4 (Frozen Tundra)**: ต้องใช้ความเร็วในการเหวี่ยงสูงเนื่องจากกลไก "Ice Bark"
- **โซน 5 (Enchanted Wilds)**: ช่วงท้ายเกมปัจจุบัน ต้องใช้ขวานระดับ Mythic
